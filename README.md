# Signup Form – The Odin Project

This project is a signup form page built as part of **The Odin Project** Foundations curriculum.

The goal of this project was to practice building structured layouts and implementing basic form validation using HTML, CSS, and a small amount of JavaScript.

---

## 📚 About The Project

This signup page was created by following a reference design provided in The Odin Project curriculum.

The exercise focuses on building a clean form layout and implementing client-side validation while maintaining good HTML structure and CSS organization.

The page includes a styled form with multiple input fields, validation feedback, and a custom layout using Flexbox.

---

## 🛠️ Built With

* HTML5
* CSS3 (Flexbox)
* Vanilla JavaScript
* Git & GitHub

---

## 🎯 What I Practiced

* Structuring form layouts with semantic HTML
* Using Flexbox for page layout and alignment
* Creating two-column form layouts without CSS Grid
* Implementing native HTML form validation
* Styling validation states with CSS (`:valid`, `:invalid`)
* Using JavaScript to validate matching passwords
* Organizing CSS for readability and maintainability
* Using Git for version control

---

## 🎨 Personal Customizations

In addition to the base exercise, I added:

* Custom phoenix/fire theme for the visual design
* Custom logo and image styling
* Styled validation feedback for form inputs
* Password confirmation validation using JavaScript
* Improved spacing and layout refinements
